# RustCraft

## Terminology

### Directions

Throughout this project the following terms are used for directions:

| terminology | definition      |
| ----------- | --------------- |
| north       | in +z direction |
| south       | in -z direction |
| west        | in -x direction |
| east        | in +x direction |
| up          | in +y direction |
| down        | in -y direction |
